ВУЗ-Assist — это PyQt6-приложение для анализа конкурсных списков и расчета проходных баллов в высших учебных заведениях. Программа предназначена для сотрудников приемных комиссий и абитуриентов для наглядного отслеживания динамики проходных баллов по образовательным программам.

Основные возможности:
- Расширенная фильтрация с операторами И/ИЛИ по всем параметрам
- Визуализация данных с графиками динамики проходных баллов и распределения
- Расчет проходных баллов по приоритетам абитуриентов
- Формирование комплексных отчетов в формате PDF
- Управление данными (CRUD-операции с базой данных)
- Генерация тестовых данных для испытаний программы
- Демонстрация пересечений абитуриентов между разными программами

Поддерживаемые образовательные программы:
- Прикладная математика (ПМ) - 40 мест
- Информатика и вычислительная техника (ИВТ) - 50 мест
- Инфокоммуникационные технологии и системы связи (ИТСС) - 30 мест
- Информационная безопасность (ИБ) - 20 мест

**Инструкция по установке/развертыванию**
Предварительные требования:

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Git (для клонирования репозитория)

**1. Установка на Windows**
Клонируйте репозиторий:

```
git clone https://github.com/yourusername/vuz-assist.git
cd vuz-assist
```

1. Создайте виртуальное окружение (рекомендуется):

```
python -m venv venv
venv\Scripts\activate
```
2.Установите необходимые библиотеки:

`pip install -r requirements.txt`
Если файл requirements.txt отсутствует, установите зависимости вручную:

`pip install pandas numpy matplotlib reportlab PyQt6`


**Установка на Linux/macOS**
1.Клонируйте репозиторий:

```
git clone https://github.com/yourusername/vuz-assist.git
cd vuz-assist
```
2.Создайте виртуальное окружение:

```
python3 -m venv venv
source venv/bin/activate
```
3.Установите зависимости:

`pip install -r requirements.txt`
**Запуск приложения**
1.Активируйте виртуальное окружение (если создавали):

Windows: `venv\Scripts\activate`

Linux/macOS: `source venv/bin/activate`

2.Запустите главный файл:

`python main.py`
Если файл называется по-другому, используйте соответствующее имя.

**Создание исполняемого файла (опционально)**
Для создания standalone-приложения используйте PyInstaller:

```
pip install pyinstaller
pyinstaller --onefile --windowed --name "VUZ-Assist" main.py
```
Исполняемый файл будет находиться в папке dist.

**Использование программы**

**Первый запуск**

1.При первом запуске база данных будет пустой.
2.Сгенерируйте тестовые данные через меню Файл → Сгенерировать данные для испытаний
3.Загрузите данные через меню Файл → Загрузить CSV (все файлы за дату)
4.Рассчитайте проходные баллы через меню Данные → Рассчитать проходные баллы

**Основные функции**

1.Фильтрация данных:
- По дате, программе, приоритету
- По диапазонам баллов (от/до)
- С логическими операторами И/ИЛИ

2.Визуализация:
- Динамика проходных баллов по дням
- Распределение баллов по программам
- Статистика приоритетов

3.Отчетность:
- Формирование PDF отчетов с проходными баллами
- Списки зачисленных абитуриентов
- Статистика по приоритетам

4.Управление данными:

- Загрузка/выгрузка CSV файлов
- Удаление дубликатов
- Коррекция данных

**Формат данных**
Программа работает с CSV файлами в формате:

Кодировка: UTF-8
Разделитель: запятая
Формат имени файла: ДД.ММ_Программа.csv (например: 01.08_ПМ.csv)

Структура CSV файла:
```
id,consent,priority,physics_score,russian_score,math_score,achievements_score,total_score
100001,1,1,85,92,88,5,270
100002,0,2,78,85,82,3,248
```

**Решение возможных проблем**

1.Проблема: Ошибка импорта библиотек
Решение:

```
# Убедитесь, что все зависимости установлены
pip install --upgrade pip
pip install pandas numpy matplotlib reportlab PyQt6
```

2.Проблема: Шрифты в PDF отчетах
Решение: Установите шрифт Arial или используйте встроенные шрифты:

Windows: Arial обычно установлен по умолчанию
Linux: Установите шрифты ttf-mscorefonts-installer

3.Проблема: Нет данных для отображения
Решение:
3.1.Сгенерируйте тестовые данные через меню "Файл"
3.2.Загрузите данные за конкретную дату
3.3.Проверьте наличие CSV файлов в папке data/

**Технические детали**

Алгоритм расчета проходного балла:

- Отбираются абитуриенты с согласием на зачисление
- Сортировка по приоритету (1-4, где 1 - наивысший)
- Внутри каждого приоритета сортировка по убыванию баллов
- Заполнение мест в порядке приоритетов
- Проходной балл = балл последнего зачисленного абитуриента

Используемые технологии:

- PyQt6 - графический интерфейс
- SQLite - база данных
- Matplotlib - визуализация данных
- ReportLab - генерация PDF отчетов
- Pandas - обработка CSV файлов
